# Recipe App

A mobile recipe application built with Flutter that allows users to browse meal categories, search for recipes, and view detailed cooking instructions.

## Features

- Browse recipe categories
- Search for dishes within categories
- View detailed recipes with ingredients and instructions
- Random recipe of the day
- YouTube video links for cooking tutorials
- Add recipes to favorites
- Receive scheduled notifications to open the app
